# KlaipedON

Pradinis `HTML + Tailwind + JS` scaffoldas būsimam KlaipedON index page.

## Paleidimas

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Projekto kryptis

- Kurdami dizainą laikomės semantinių sekcijų
- Vengiame perteklinės JS logikos
- Iš anksto galvojame apie WordPress partialus ir ACF laukus
